# DSL 内嵌政策摘要

这些内容来自 Demo Code 节点，不代表正式越南政策。

## 禁寄与限制寄递

- 规则会因国家/地区、渠道和物品类型变化。
- 液体、电池、食品、粉末、药品通常需要进一步核实。
- 最终是否收寄，以当地网点或系统校验为准。

## 时效

- 始发地、目的地、天气、分拣中转、清关和末端派送均可能影响时效。
- 有运单号时先查询实时轨迹，再判断是否异常。

## 理赔

- 破损或遗失通常需要运单号、问题描述、照片或价值证明。
- 客服只能先登记核实，不能承诺赔付结果。

## 改址

- 是否支持取决于实时物流状态。
- 已签收或派送中的包裹通常不支持直接修改。
- 运输中可以尝试申请，最终以系统和网点处理结果为准。
